Filter Settings

August 14, 2014 by ptguy·0 Comment

This tab provides a set of options to query your desired content easily.

Content type

This option used to select type of content you want to display. It contains Post, Page and all custom post types (WooCommerce products, portfolios…)

Common filters

This section contains common options to customize the query.

Sticky posts

By default, sticky posts are included at the beginning of posts returned. Checking this option to prevent that action.

In list

This option used to display some specific posts quickly, by entering their IDs.
It also allows you to restrict what post should be displayed (a post is displayed only if its ID is equal to any of the values IN this list).

Excludes

This option allows you to restrict what post should be displayed directly.
When you preview the output of View, if you don’t want to show any post, just get its ID and enter to this field.

Limit

Set the limit of total number of posts you want to show in this View.
If pagination is enabled, this number is total of posts on all pages.

Offset

This option used to pass over a number of posts from the beginning of result set.

Advanced filters

This section contains all advanced options to query your desired content.

Taxonomy (Categories, Tags…)

This group helps you to filter content by taxonomies and terms.
By default, if you are selecting “Post” as “Content type”, you will able to filter posts by Category, Tag, Post Format in this group.
To filtering posts by Category, check “Category” checkbox. It will show “Category Settings” box in “Terms” panel.

“In”

To filter posts by specific categories, just type their names to the “In” field. If you want to remove selected category, just click on “x” icon like image below

There are 2 options which helps you to quick select name of categories:

  • Start with: Enter a character to get all terms which their name starts with it (non case-sensitive)
  • End width: Enter a character to get all terms which their name ends with it (non case-sensitive)

“Not in”

To hide all posts associate with specific categories from output, just type their names to the “Not in” field.

Order & Orderby

This group helps you to order items in output by their Name/Title, Date… in Ascending/Descending (ASC/DESC) order.

There are a lot of options here, to understand and know how to use them, please check these “How to” posts

Author

This group helps you to filter content by author.

To get content of specific authors, just enter their username in “Written by” field.
To hide all content of specific authors from output, just enter their username in “Not written by” field.

Status

This group helps you to filter content by their status: Publish, Private, Draft…

Search

This group helps you to find content which contains a keyword, it works like the Search function in front-end of your website.
