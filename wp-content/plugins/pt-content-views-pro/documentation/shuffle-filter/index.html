Shuffle Filter

August 16, 2014 by ptguy·0 Comment

You have list of awesome portfolios. What is best way you can display them?
Well, you should select Grid or Pinterest as “View type” and check this option “Enable reorder and filter posts with a nice shuffling animation”.

There are some options for you to customize the filter bar.

Type

There are currently 3 different output for filter bar. You can change color and background of each filter option in “Style Settings” tab.

Position

Easy to align the filter bar in left, center or right side.
