What is Content Views?

August 13, 2014 by ptguy·0 Comment

Overview

Content Views is a free WordPress plugin which helps you to display posts, pages in responsive Grid layout without coding. It helps you to display posts in your Homepage, in a custom page, in a Text widget or any where you can imagine. All what you have to do is just create a View to query your desired posts through an intuitive form (with Live preview) and paste the generated shortcode to right place. No coding, no hassle.

Content Views Pro is a premium WordPress plugin, which adds more powerful features that helps you to display any kind of post types (post, page, WooCommerce product, Portfolio…) in most beautiful layouts you’ve ever known: Grid, Pinterest, FB Timeline, Scrollable list, Collapsible list.

Compatibility

Both Content Views and Content Views Pro requires WordPress 3.3 or higher.
Normally, they work well with all WordPress frameworks (Free: Carrington, Gantry, Hybrid, Options Framework… Commercial: Genesis Framework, WP Canvas, IStackpress, K9 Canvas…) and themes.
Sometimes it has style problem after install CV plugin, when style (Bootstrap) of CV overwrites current theme’s style. But please don’t worry, it always takes only some minutes to fix these problem.

View

View is a custom post type which Content Views plugin uses to store options to produce output for each time you need to display your content.

How can I use it?

Assuming you want to display latest posts in your home page, display posts in a specific category in a page… Can Content Views plugin help? Is it easy to do?

The answer for both questions is YES. You can do all your needs which relates to display content with Content Views easily, with 3 simple steps:

Step 1 : Filter your desired content (post, page, product, portfolio…) by author, category, tag…
Step 2 : Choose the layout you want to display your content and control output more details (if you want)
Step 3 : Save View then paste the generated shortcode [pt_view id="UNIQUE_ID"] to right place (to editor of a page, to a Text widget... anywhere you want to display your posts). If you are a developer, you can <?php echo do_shortcode('[pt_view id="UNIQUE_ID"]'); ?> in your WordPress theme.

That’s it.
