The Settings page

August 17, 2014 by ptguy·0 Comment

After you installed Content Views plugin, you will see “Content View Settings” in left menu of your Admin dashboard.
That menu will navigate you to setting page of Content Views.
With Content Views Pro plugin installed, you will see:

  • current versions of PRO, Free version
  • link to our Member-only forum
  • Frontend assets: There are options where you can check/uncheck to load/unload style and script of libraries (Bootstrap, Colorbox). Each option should be check/uncheck in a specific case, please read description on each option to make right decision.

User role who can manage Views

By default, Administrator and Editor can manage (add, edit, trash) Views. This option helps you to manage what role can manage Views.
If you want to allow users with Editor role can manage Views, you can leave this option as default, or select “Editor” option in the select box.
If you don’t want anyone else but you can manage Views . You should select “Administrator” option in select box.

License key

With license key, you can update Content Views Pro and register an account to our support system.
After you purchased Content Views PRO, you will receive an email which includes:

  • link to a page where you will see your License key and download button (with all versions so far)
  • link to register page
