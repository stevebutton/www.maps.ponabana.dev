Views management (Add, Edit, Search)

August 17, 2014 by ptguy·0 Comment

You may create a lot of View to display your content in different positions and layouts in your website. So where to find all Views you created, how to search for a View to edit, delete?
In left menu, hovering on “Content View Settings”, you will see 3 sub menus:

  • Content View Settings
  • All Views
  • Add New

The first sub item will navigate you to Settings page, which is introduced here. Let’s explore 2 other sub menus.

All Views

This sub menu will navigate you to a page which list all created Views.

Each View name will contain View title (which you entered in Add/Edit View page) and View ID (an auto generated string).
If you want to search for an existed View, just type its name or ID to the text field then hit “Search Views” button.
By hovering on View name, you will see available actions like below image

Edit View

By clicking on “Edit” link, it will navigate you to Edit View page, where you can edit stored options of a View.

Duplicate View

Sometimes you want to reuse some options of an existed View, this is the easiest way you can do.
By clicking on “Duplicate” link, it will auto create a new View which inherits all stored options of selected View.

Trash View

If you don’t want to use a View anymore, just click on “Trash” button. But please be careful when do this action, make sure that View is not used in any other place of your site.

Add New

This link will navigate you to Add View page, where has tons of options for you to query and display your content in your own way.

At top of Add View page, there is Preview panel where you can see the output on the fly. Below it is “View title” field, you should enter title for each View you create to identify them more easily in All Views page. It doesn’t need to be complex, just somethings like: Latest posts at Homepage, Most popular posts in News page, For Portfolio page…

For more details about all options in Add View page, you can start with Filter Settings post.
